# ChoreTracker

ChoreTracker is a simple web application built with **C#** , **ASP.NET MVC** , and **Entity Framework** to help manage and track household chores. This project demonstrates basic CRUD (Create, Read, Update, Delete) operations within the MVC architecture, using a SQL Server database.

## Features

- User registration and login
- Create, view, edit, and delete chores
- Assign chores to users
- Cancel chore assignments
- Session management for user authentication

## Technologies Used

- **C#**
- **ASP.NET Core MVC**
- **Entity Framework Core**
- **SQL Server**

## Migrations

Migrations are used to manage database schema changes, including allowing nullable values for `WorkerId` and `UserId` in the `Jobs` table.

## Usage

- Register or log in.
- View and manage chores from the dashboard.
- Create new chores, edit existing ones, or assign chores to users.
- Delete or cancel chores as needed.
